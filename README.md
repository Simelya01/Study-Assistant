O Study Assistant é uma aplicação desenvolvida em Python com interface gráfica, criada a partir da minha necessidade de organizar os estudos para o vestibular.

A ideia surgiu da dificuldade em controlar tarefas, prazos e conteúdos no dia a dia, então desenvolvi uma solução simples para ajudar nesse processo. O sistema permite cadastrar e organizar atividades de estudo de forma prática e intuitiva.

Esse projeto representa minha evolução na programação, aplicando na prática conceitos de lógica, manipulação de dados e desenvolvimento de interfaces.

⸻

🛠 Tecnologias Utilizadas

* Python
* Tkinter
* JSON
